## 🎯 核心配色方案

### 主色系定義
```
深藍色 (Primary):   #1e40af   用於主要標題、主按鈕、邊框
青色 (Accent):      #0891b2   用於強調、次要元素、懸停效果
亮青色 (Light):     #06b6d4   用於成功提示、亮度調度
```

### 推薦組合
- **背景**: `linear-gradient(135deg, #1e40af 0%, #0891b2 100%)`
- **按鈕**: `linear-gradient(135deg, #1e40af, #0891b2)`
- **懸停**: 轉換至青色或亮青色
