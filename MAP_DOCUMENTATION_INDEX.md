## 🔑 關鍵信息速查

### 功能特點
- 🗺️ 互動式地圖視圖
- 📍 監視器位置標記
- 🎨 顏色編碼（TDX 藍色，其他橙色）
- 💬 彈窗詳細信息
- 🔄 篩選條件同步
- 📊 自動視野調整

### 技術棧
- 🎯 Leaflet.js v1.9.4 地圖庫
- 🌍 OpenStreetMap 圖層
- 🎨 Font Awesome 圖標
- 💻 原生 JavaScript（無框架）

### 性能指標
- ⚡ 地圖初始化：< 1 秒
- 📦 標記加載（50 個）：< 500ms
- 🔄 篩選更新：< 1 秒
- 📈 記憶體占用：5-10 MB

### 瀏覽器支持
- ✅ Chrome / Firefox / Safari / Edge
- ⚠️ IE 11 部分支持
