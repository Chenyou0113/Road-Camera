# 省道監視器 - 直接顯示畫面更新

## 🎯 更新內容

已修改 `road.html` (省道監視器頁面) 使相機畫面直接以大尺寸顯示。

### 📐 具體改動

#### 1. **影像高度增加**
   - 之前：200px
   - 現在：**400px** (2倍增加)

#### 2. **網格佈局調整**
   - 之前：3 列 (3 個監視器並排)
   - 現在：**2 列** (2 個監視器並排)
   - 原因：讓每個監視器有更多空間展示 400px 高的影像

#### 3. **卡片信息優化**
   - 信息區域高度：15px → **12px** (更緊湊)
   - 標題字號：24px → **16px**
   - 間距調整：更適合大圖片卡片

#### 4. **互動效果改進**
   - 移除了圖片上的黑色覆蓋層 (overlay)
   - 調整 hover 縮放：1.05 → **1.02** (微妙的效果)
   - 卡片陰影增強：0 5px 15px → **0 8px 20px**

### 📱 響應式適配

```
桌面 (1200px+)   : 2 列 × 400px 高影像
平板 (768-1200px): 2 列 × 400px 高影像  
手機 (768px以下) : 1 列 × 400px 高影像
```

### ✨ 效果

- ✅ 省道監視器影像直接顯示，清晰可見
- ✅ 無需點擊彈窗即可看到監視器畫面
- ✅ 提供更好的實時監控體驗
- ✅ 保持點擊卡片可放大查看的功能（模態框仍存在）

### 🖼️ 視覺對比

**之前：** 小卡片 (200px) + 標題 + 信息
**現在：** 大監視器畫面 (400px) + 簡潔標題 + 信息
